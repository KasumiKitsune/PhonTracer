## 📌 项目简介

PhonTracer 是一个专为语音学、方言学和声调声学分析设计的轻量级桌面工具。它的**唯一核心功能**是：**将输入的语音音频，自动化、批量地转换为提取好元音核心（Vowel Nucleus）声调（基频）特征的结构化数据格式。**

无论你是处理整段的长录音，还是已经切分好的独立字音文件，本工具都能帮助你快速定位有效发音段，提取基频（F0）数据，并导出为标准格式供后续的统计或绘图使用。

## 🚀 核心工作流程

工具提供两种主要的工作模式，以适应不同的田野调查或语料整理习惯：

### 模式一：单条长音频切分
适用于一次性录制了整张字表、未做后期剪辑的超长音频。

1. **导入长音频**：加载完整录音文件（支持 `.wav` / `.mp3`）。
2. **文本匹配与粗切分**：
   - 粘贴字表文本。
   - 程序将执行宏观语音端点检测（VAD），结合字表顺序自动切出包含发音的片段。
3. **可视化微调（可选）**：通过“可视化手动切分”界面，对自动识别的边界进行拖拽微调，确保片段精准包裹目标音节。
4. **特征提取与导出**：应用算法精准提取元音段并导出数据。

### 模式二：多条独立音频批量提取
适用于已经将每个字/词剪辑为独立文件的语料库。

1. **导入独立音频**：批量选中所有单字音频文件。
2. **自动映射字表**：
   - **模糊匹配**（推荐）：程序根据音频文件名，智能识别并映射字表中的对应单字。
   - **顺序匹配**：按照文件导入顺序与字表列表强制对应。
3. **特征提取与导出**：利用多进程加速，一并完成所有音频文件的声调特征提取。

## ⚙️ 核心参数控制

为了保证提取出的声调数据具有严谨的声学参考价值，工具提供了直观的参数调优：

- **等分点 (N)**：在一个提取出的元音段落内，均匀采样基频的次数（默认为 11 点），这是绘制归一化声调曲线的基础。
- **能量落差 (dB)**：算法基于最大能量点向两侧寻找边界，该值设定了允许的能量下降阈值（决定了提取出的元音边界的宽窄）。
- **最短时长**：过滤极短的突发噪声，保证提取片段的有效性。
- **边缘静音裁切**：自动忽略音频两端低于 -50dB 的绝对静音区域，让算法更聚焦于核心波形。

## 📊 输出数据

所有流程的最终目的都是输出高度标准化的文本/表格数据。导出的数据主要包含：
*   **元数据**：单字名称、所属声调组别。
*   **时间对齐**：在原始音频中的起止时间点。
*   **声学特征**：等分采样提取出的 Pitch (F0) 数值序列。

该格式可无缝导入 Excel、R、Python (Pandas) 中，用于声学统计分析和传统五度值声调格局图的绘制。

## 🛠️ 本地运行

本项目基于 Python 构建，界面使用 CustomTkinter，音频分析引擎基于 `praat-parselmouth`。

```bash
# 1. 克隆仓库
git clone https://github.com/KasumiKitsune/Tone_extractor.git
cd Tone_extractor

# 2. 创建并激活虚拟环境 (推荐)
python -m venv .venv
# Windows:
.\.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动程序
python main.py
```
